# Backend 개발 명세서

## 1. 기술 스택

- **Framework**: Spring Boot 3.2
- **언어**: Java 17
- **데이터베이스**: MariaDB
- **빌드**: Maven

## 2. 프로젝트 구조

```
backend/
├── src/main/java/com/smartlearning/
│   ├── controller/     # REST API
│   ├── service/        # 비즈니스 로직
│   ├── repository/     # 데이터 액세스
│   ├── entity/         # JPA 엔티티
│   ├── dto/            # 데이터 전송 객체
│   └── config/         # 설정
├── src/main/resources/
│   └── application.yml
└── pom.xml
```

## 3. 주요 설정

```yaml
# application.yml
server:
  port: 8080

spring:
  datasource:
    url: 
    username: 
    password: 
  jpa:
    hibernate:
      ddl-auto: update

jwt:
  secret: 
  expiration: 86400000
```
